FTLabAct3: Simple Calculator using HTML, CSS, and JS

Objective:

Student will create a functional and simple calculator. Also, this activity lets the students to:

  • Implement Advanced CSS for layout design and transitions
  • Use JavaScript to handle basic arithmetic operations such as addition, subtraction, multiplication, and division

Instructions:

  1. This activity should be done individually.
  2. Use separate files for content, presentation, and behavior.
  3. Use either flexbox or grid to layout the buttons in the calculator.
  4. The buttons should have hover effects. Any transition property will do.
  5. Read the criteria for grading carefully.

FAQs:

  • You are not required to copy the example above as your output.
  • You can freely design your calculator.
  • You can search for photo inspo for the layout and design.
  • You can use color palette generator websites.
  • Do not use inappropriate images and text.
  • Do not COPY/REUSE/RECYCLE your classmate's code.

Criteria for Grading:

  • Layout: 25%
    • Effective use of Flexbox/Grid properties.
    • Proper alignment and spacing of buttons.
  • Functionality: 50%
    • The calculator should perform basic arithmetic such as addition, subtraction, multiplication, and division.
  • Design: 25%
    • Consistent and visually appealing use of colors, fonts, and spacing.
    • Effective use of hover and transitions on buttons.
